Randomly choose 10 leaves on this plant. For each leaf, what is the estimated percent of leaf area that has visible ozone damage?

Use the injury scale and accompanying photos on the next page to categorize the injury to the best of your ability, recording the index value in the table below the leaf photos. The leaf number is only used to identify that there are different leaves, so the order or number does not matter. Be careful to collect data from each leaf only once.
